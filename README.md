# Mavi Zuccaciye - Stüdyo Fotoğraf Üretici

Ürün fotoğrafını yükle → Gemini AI ile stüdyo görseli üret → Canva'ya otomatik yükle

## Kurulum

### Environment Variables (Railway'de ekle)
```
GEMINI_API_KEY=your_gemini_api_key
CANVA_API_KEY=your_canva_api_key
CANVA_DESIGN_ID=DAHCFrUWEbo
```

### Deploy
1. Bu repoyu GitHub'a yükle
2. Railway'de "New Project" → "Deploy from GitHub"
3. Environment variables ekle
4. Deploy!

## Kullanım
1. Web arayüzüne git
2. Ürün fotoğrafı yükle
3. "Stüdyo Fotoğrafı Oluştur" butonuna bas
4. Görsel otomatik Canva'ya yüklenir
